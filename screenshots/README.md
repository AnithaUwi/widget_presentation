# Screenshots Folder

This folder should contain your demo screenshot.

## To Add Screenshot:

1. Run the app on your emulator or device
2. Take a screenshot showing the recipe cards
3. Save it as `demo_screenshot.png` in this folder
4. The README.md will automatically display it

## Screenshot Requirements:
- Show the full app with control panel and recipe cards
- Make sure the Card properties are clearly visible
- Recommended size: 1080px width or higher
- Format: PNG or JPG

## Quick Screenshot Tips:
- **Android Emulator**: Click the camera icon in the toolbar
- **iOS Simulator**: Cmd+S (Mac) or use Device → Screenshot
- **Physical Device**: Use device screenshot feature, then transfer to computer

After adding the screenshot, commit it with:
```bash
git add screenshots/demo_screenshot.png
git commit -m "Add demo screenshot for README"
git push
```
